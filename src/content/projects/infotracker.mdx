InfoTracker is an web app designed to demonstrate what information a browser can access from your system without requiring additional permissions.

No server is connected to this app, so only client-side methods are shown.

It retrieves and displays various data points, including:

- Installed fonts
- FPS
- Local storage data
- Battery information
- Screen size and resolution
- Device motion and orientation data
